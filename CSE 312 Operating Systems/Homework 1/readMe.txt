To run the program:
	make command must be entered to the terminal.
	mykernel.iso must be added to operating system in the virtualbox.
	make command must be entered again to the terminal.
To clean object files:
	make clean command must be entered to the terminal.
To start init process:
	press 'a' on the keyboard to start the init process.
Display processes:
	press 'p' on the keyboard to print the processes.
