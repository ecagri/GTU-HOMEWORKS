To compile:
    Enter make command to terminal.
To run:
    Enter 
    ./BibakBOXServer foldername_which_must_be_exist_in_current_directory thread_pool_size port_number for server.
    ./BibakBOXClient foldername port_number for client.
    Server must be start first.
To clean:
    Enter meke clean command to terminal.
